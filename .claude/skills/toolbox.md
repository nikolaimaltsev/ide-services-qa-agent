# Toolbox App

*If GUI interaction is required, read `.claude/skills/computer-use.md` first*

- Toolbox App: lives in the system tray; use arrow keys / Enter / Tab inside it
- IDE Services server URL: Toolbox Settings → IDE Services
- Typical flow to connect IDE Services to Toolbox described here: https://www.jetbrains.com/help/ide-services/connect-to-instance.html#join-organization
- JetBrains Toolbox App docs: https://www.jetbrains.com/help/toolbox-app/installation.html
